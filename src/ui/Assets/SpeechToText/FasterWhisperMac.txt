Faster Whisper Mac runs the "faster-whisper" Python package (https://github.com/SYSTRAN/faster-whisper)
from the command line on macOS (Apple Silicon and Intel).

Installation:

        pip3 install faster-whisper

Subtitle Edit detects the installation by running: python3 -c "import faster_whisper"

Models:

        tiny, base, small, medium, large-v2, large-v3

        Models are downloaded automatically by faster-whisper on first use and cached in
        ~/.cache/huggingface/hub - no manual model download is needed.

Device:

        Subtitle Edit passes --device auto so CTranslate2 picks the fastest available
        backend on the Mac (Apple Silicon acceleration where supported, otherwise CPU).

Command line built by Subtitle Edit:

        faster-whisper --device auto --output_format srt --output_dir <audio folder>
                       --language <language> --model <model> [extra parameters] <audio file>

        The SRT result is written next to the input audio file and loaded automatically.
        Translation to English uses: --task translate

Common extra parameters (add in the parameters text box):

        --compute_type {default,auto,int8,int8_float16,int16,float16,float32}
        Type of quantization to use (default: auto)
        --threads THREADS     Number of threads used for CPU inference (default: 0)
        --task {transcribe,translate}
        Whether to perform X->X speech recognition ('transcribe') or X->English translation
        ('translate') (default: transcribe)
        --temperature TEMPERATURE
        Temperature to use for sampling (default: 0)
        --best_of BEST_OF     Number of candidates when sampling with non-zero temperature (default: 5)
        --beam_size BEAM_SIZE
        Number of beams in beam search; only applicable when temperature is zero (default: 5)
        --initial_prompt INITIAL_PROMPT
        Optional text to provide as a prompt for the first window (default: None)
        --condition_on_previous_text CONDITION_ON_PREVIOUS_TEXT
        If True, provide the previous output of the model as a prompt for the next window (default: True)
        --word_timestamps WORD_TIMESTAMPS
        (Experimental) Extract word-level timestamps and refine the results based on them (default: False)
        --vad_filter VAD_FILTER
        Enable voice activity detection (VAD) to filter out parts of the audio without speech (default: False)
        --vad_threshold VAD_THRESHOLD
        When vad_filter is enabled, probabilities above this value are considered as speech (default: None)
